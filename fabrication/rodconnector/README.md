# Rodconnector
3D printable file for connecting two rods at a flexible angle. As of 20-02-2023 these can only connect 12mm rods.

![](https://i.imgur.com/P6KwMfc.png)

## Details
|Description|Value|
|---|---|
|Material| PLA
|Used filament| 7.3g
|Nozzle size| 0.4mm
|Laminar thickness| 0.2mm
|Infill| 20%
|Time to print| 00:53
